Xalior-core
===========

Xalior's base Minecraft Library
-------------------------------

This library provides the core functionality requires by Xalior's other mods. It includes:

  * World-Gen
    * Xalorium Ore - this adds Xalorium throughtout the world.
  * New Craftable Items
    * Xalorium Dust
    * Xalorium Bars
    * Xalorium Blocks
    * Xalorium Lights
